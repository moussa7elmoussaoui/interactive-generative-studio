About

Interactive Generative Studio

Interactive Generative Studio is a university term project that demonstrates full-stack Flask development, creative coding, object-oriented programming, image processing, artistic data visualization, and responsive web design.

The application is built using Python 3.12, Flask, Jinja2, Pillow, OpenCV, Pandas, NumPy, Matplotlib, Scikit-learn, Flask-WTF, HTML5, CSS3, Vanilla JavaScript, and the Canvas API.

Users can generate procedural artwork, transform images, create artistic visualizations from datasets, and browse all generated creations through an interactive gallery.
